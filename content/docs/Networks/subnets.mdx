#### Terminology

<Callout>CIDR (Classless Inter-Domain Routing)** notation was developed to provide more granularity than legacy classful addressing. CIDR
notation is expressed as /XX.</Callout>
<Callout>Wildcard Mask** indicates which parts of an IP address are available for examination.</Callout>
<Callout>Binary Subnet Mask** provides a clear representation of the subnet mask in binary format, illustrating the network and host portions.</Callout>
<Callout>IP Address Classes**: Classes A, B, and C are used for unicast addressing. Class D is reserved for multicast, and Class E is reserved for experimental purposes.</Callout>
<Callout>Localhost (127.0.0.0/8)** is used for loopback testing within a host.</Callout>
<Callout>Zeroconf/Automatic Private IP Addressing (APIPA)**: 169.254.0.0/16 is used for self-assigned addresses when DHCP fails.</Callout>
