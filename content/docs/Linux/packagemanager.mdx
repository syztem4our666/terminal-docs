### Package Management Commands

#### Install from Repository

#### Snap Package Manager
```bash
snap install [packageName]
```

#### Flatpak Package Manager
```bash
flatpak install [remote] [packageName]
```

#### APT (Advanced Package Tool)
```bash
apt install [packageName]
```

#### YUM (Yellowdog Updater Modified)
```bash
yum install [packageName]
```

#### DNF (Dandified YUM)
```bash
dnf install [packageName]
```

#### PKG (FreeBSD Package Manager)
```bash
pkg install [packageName]
```

#### APT-GET (Older Debian-Based Systems)
```bash
apt-get install [packageName]
```

#### Pacman (Arch Linux)
```bash
pacman -S [packageName]
```

#### Pip (Python Package Installer)
```bash
pip install [packageName]
```

#### Zypper (openSUSE)
```bash
zypper install [packageName]
```

### Install from File

#### Flatpak File
```bash
flatpak install [package.flatpakref]
```

#### Debian Package
```bash
dpkg -i [package.deb]
```

#### RPM Package
```bash
yum install [package.rpm]
```

#### DNF RPM Package
```bash
dnf install [package.rpm]
```

#### TXZ Package (FreeBSD)
```bash
pkg add [package.txz]
```

#### Tarball Package (Arch Linux)
```bash
pacman -U [package.tar.gz]
```

#### Local Python Package
```bash
pip install [/path/to/package]
```

### Add Repository Source

#### APT (Advanced Package Tool)
```bash
add-apt-repository ppa:[userName]/[ppaName]
```

#### APT (Advanced Package Tool)
```bash
add-apt-repository deb [url] [derivative]
```

#### YUM (Yellowdog Updater Modified)
```bash
yum-config-manager --add-repo [url]
```

#### DNF (Dandified YUM)
```bash
dnf config-manager --add-repo [url]
```

#### Zypper (openSUSE)
```bash
zypper addrepo [url] [alias]
```
